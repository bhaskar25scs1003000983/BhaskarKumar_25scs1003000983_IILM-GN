# Climate Data Analysis and Visualization using Python

This project analyzes historical global temperature data (1750–2015) using Python and exploratory data analysis techniques.

## Features
- Data preprocessing and cleaning
- Missing value handling
- Correlation heatmap visualization
- Seasonal decomposition analysis
- Yearly temperature trend analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels

## Files
- `GlobalTemperaturesHeatmap.ipynb` → Main notebook
- `AIML Project Report.pdf` → Detailed project report
